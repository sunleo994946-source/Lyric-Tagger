<a name="english"></a>
## English

A web-based lyric timing tool for creating subtitle files.

### Features

- Import music files (MP3, WAV, etc.)
- Import lyrics from TXT files
- Press SPACE to tag timing for each lyric line
- Click on any line to jump to that time
- Export as SRT subtitle file
- Multi-language support (Chinese, English, Japanese)

### How to Use

1. Click "Import Music" to select your audio file
2. Click "Import Lyrics" to select your TXT lyrics file
3. Play the music and press SPACE when you hear each lyric
4. Click on any line to re-timing if needed
5. Click "Export SRT" to save your subtitle file

### Feedback

If you encounter any bugs or errors, please [open an issue](https://github.com/anomalyco/opencode/issues).

## 日本語

歌詞タイミングツール - サブタイトルファイル作成用

### 機能

- 音楽ファイルのインポート（MP3、WAVなど）
- TXTファイルから歌詞をインポート
- スペースキーで各歌詞行のタイミングを記録
- 行をクリックしてその時間にジャンプ
- SRTサブタイトルファイルとしてエクスポート
- 多言語対応（中国語、英語、日本語）

### 使い方

1. 「音楽をインポート」で音声ファイルを選択
2. 「歌詞をインポート」でTXTファイルを選択
3. 音楽を再生し、歌詞が聞こえたらスペースキーを押す
4. 必要に応じて行をクリックしてタイミングを修正
5. 「SRTをエクスポート」で字幕ファイルを保存

<a name="中文"></a>
## 中文

歌词打轴工具 - 用于创建字幕文件

### 功能

- 导入音乐文件（MP3、WAV等）
- 从TXT文件导入歌词
- 按空格键为每行歌词打轴
- 点击任意行跳转到该时间点
- 导出SRT字幕文件
- 多语言支持（中文、英文、日文）

### 使用方法

1. 点击「导入音乐」选择音频文件
2. 点击「导入歌词」选择TXT歌词文件
3. 播放音乐，听到歌词时按空格键
4. 如需修改，点击歌词行可重新打轴
5. 点击「导出 SRT」保存字幕文件
